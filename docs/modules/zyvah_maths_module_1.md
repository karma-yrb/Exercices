# Draft - Zyvah Maths Module 1

## Meta
- Theme: Studio Maths - priorites operatoires
- Objectif: parentheses, multiplication/division, lecture gauche->droite
- Duree cible: ~10 min par mission

## Mission 1 - L'Ordre du Mixage

### Objectifs pedagogiques et competences
- Appliquer la priorite des parentheses.
- Calculer des expressions imbriquees.

### Index technique (ordre des steps)
1. msg - Le Code du Studio
2. quiz - Test Cabine 1
3. quiz - Test Cabine 2
4. write - Saisie de Gain
5. msg - L'Imbrication (Double Deck)
6. write - Deep Mix
7. write - Deep Mix Final
8. quiz - Le Piege du Master
9. write - Mastering
10. challenge - Session Finale Jour 1

### Ecran 1 - msg - Le Code du Studio
- Texte: "Zyvah, pour que le son soit bon, il y a un ordre. En maths, c'est pareil : on appelle ca les Priorites."
- Encadre: "1. Les Parentheses ( ) : On s'en occupe TOUJOURS en premier. C'est le signal direct qui ne doit pas etre filtre."

### Ecran 2 - quiz - Test Cabine 1
- Question: "Dans le calcul 5 + (2 x 3), par quoi commences-tu ?"
- Options: "5 + 2" / "2 x 3" / "5 + 3"
- Reponse: "2 x 3"
- Feedback: "Exact ! Le Master ( ) passe avant tout le reste."

### Ecran 3 - quiz - Test Cabine 2
- Question: "Calcule le contenu de la parenthese : 10 - (2 + 3)"
- Options: "5" / "8" / "11"
- Reponse: "5"
- Feedback: "Exact. 2 + 3 = 5."

### Ecran 4 - write - Saisie de Gain
- Question: "Calcule le resultat final de : 10 - (2 + 3)"
- Reponse attendue: "5"
- Hint: "Tu as deja trouve 5 pour la parenthese. Fais maintenant 10 - 5."
- Feedback: "Master ( ) gere. 10 - 5 = 5."

### Ecran 5 - msg - L'Imbrication (Double Deck)
- Texte: "Parfois, tu as des parentheses dans des parentheses. On commence par la plus profonde."
- Encadre: "10 + [ 2 x (3 + 1) ] -> (3 + 1) = 4 -> [ 2 x 4 ] = 8 -> 10 + 8 = 18."

### Ecran 6 - write - Deep Mix
- Question: "Calcule la parenthese interieure : 5 x (2 + (3 - 1))"
- Reponse attendue: "2"
- Hint: "Regarde bien le moteur le plus au centre : (3 - 1)."
- Feedback: "Bien ! 3 - 1 = 2."

### Ecran 7 - write - Deep Mix Final
- Question: "Maintenant, calcule le total : 5 x (2 + 2)"
- Reponse attendue: "20"
- Hint: "Priorite a la parenthese (2 + 2), puis multiplie le resultat par 5."
- Feedback: "5 x 4 = 20. Mixage propre."

### Ecran 8 - quiz - Le Piege du Master
- Question: "Que vaut 20 - (5 + 5) ?"
- Options: "20" / "10" / "15"
- Reponse: "10"
- Feedback: "On fait 5+5=10 d'abord. 20-10=10."

### Ecran 9 - write - Mastering
- Question: "Calcule : (15 - 5) x 2"
- Reponse attendue: "20"
- Hint: "D'abord la soustraction dans les parentheses, puis multiplie par 2."
- Feedback: "10 x 2 = 20."

### Ecran 10 - challenge - Session Finale Jour 1
- Question: "Calcule ce mix complet : (3 + 7) x (2 + 2)"
- Reponse attendue: "40"
- Hint: "Calcule chaque parenthese separement, puis multiplie les deux resultats entre eux."
- Feedback: "10 x 4 = 40. Mixage parfait !"

## Mission 2 - Multiplication & Gain

### Objectifs pedagogiques et competences
- Prioriser multiplication/division sur addition/soustraction.
- Lire les operations de gauche a droite a priorite egale.

### Index technique (ordre des steps)
1. msg - Le Gain de Sortie
2. quiz - Reglage Niveau 1
3. write - Gain Rapide
4. quiz - Reglage Niveau 2
5. write - Division de Flow
6. msg - Melange Complexe
7. quiz - Test Gain
8. write - Mixage Gain
9. write - Gain Maximum
10. challenge - Boss de Session 2

### Ecran 1 - msg - Le Gain de Sortie
- Texte: "Apres les parentheses, les multiplications (x) et divisions (/) sont plus fortes que les additions et soustractions."
- Encadre: "2 + 5 x 2 = ? -> d'abord 5 x 2 = 10 -> puis 2 + 10 = 12."

### Ecran 2 - quiz - Reglage Niveau 1
- Question: "Calcule : 10 - 2 x 4"
- Options: "32" / "2" / "8"
- Reponse: "2"
- Feedback: "Bien ! 10 - 8 = 2. La multiplication est prioritaire."

### Ecran 3 - write - Gain Rapide
- Question: "Calcule : 3 + 4 x 2"
- Reponse attendue: "11"
- Hint: "Le gain (x) passe avant l'ajout (+). Fais 4 x 2 d'abord."
- Feedback: "4 x 2 = 8, puis 3 + 8 = 11."

### Ecran 4 - quiz - Reglage Niveau 2
- Question: "Calcule : 15 / 3 + 2"
- Options: "7" / "3" / "5"
- Reponse: "7"
- Feedback: "15/3 = 5, puis 5 + 2 = 7."

### Ecran 5 - write - Division de Flow
- Question: "Calcule : 20 - 10 / 2"
- Reponse attendue: "15"
- Hint: "La division (/) est prioritaire sur la soustraction (-)."
- Feedback: "10 / 2 = 5, 20 - 5 = 15."

### Ecran 6 - msg - Melange Complexe
- Texte: "Multiplications et divisions ont la meme force. Si elles sont cote a cote, on suit le flow de lecture."

### Ecran 7 - quiz - Test Gain
- Question: "Calcule : 6 + 4 x 3 - 2"
- Options: "28" / "16" / "22"
- Reponse: "16"
- Feedback: "4x3 = 12. Ensuite 6 + 12 - 2 = 16."

### Ecran 8 - write - Mixage Gain
- Question: "Calcule : 5 x 4 / 2"
- Reponse attendue: "10"
- Hint: "x et / ont la meme force, lis de gauche a droite : 5 x 4, puis divise par 2."
- Feedback: "20 / 2 = 10."

### Ecran 9 - write - Gain Maximum
- Question: "Que vaut 100 - 50 / 2 ?"
- Reponse attendue: "75"
- Hint: "Diviser avant de soustraire ! Calcule 50 / 2 d'abord."
- Feedback: "50/2 = 25. 100 - 25 = 75."

### Ecran 10 - challenge - Boss de Session 2
- Question: "Calcule : 10 + 10 x 10"
- Reponse attendue: "110"
- Hint: "Multiplie d'abord les deux 10, puis ajoute le premier 10."
- Feedback: "Puissance maximum !"

## Mission 4 - Parentheses & Effets

### Objectifs pedagogiques et competences
- Utiliser les parentheses pour changer l'ordre des calculs.
- Comparer deux expressions proches.

### Index technique (ordre des steps)
1. msg - Forcer le Mix
2. interactive - Comparaison de Signal
3. write - Calcul Master
4. challenge - Boss de Session 4

### Ecran 1 - msg - Forcer le Mix
- Texte: "Si tu veux qu'une addition soit faite AVANT une multiplication, tu dois utiliser les parentheses."
- Encadre: "(2 + 3) x 2 = 5 x 2 = 10"

### Ecran 2 - interactive - Comparaison de Signal
- Question: "Lequel de ces mix donne 20 ?"
- Options: "(5 + 5) x 2" / "5 + 5 x 2"
- Reponse: "(5 + 5) x 2"
- Feedback: "Oui ! Avec parentheses = 10x2=20. Sans = 5+10 = 15."

### Ecran 3 - write - Calcul Master
- Question: "Que vaut (20 - 10) / 2 ?"
- Reponse attendue: "5"
- Hint: "La parenthese d'abord !"
- Feedback: "10 / 2 = 5."

### Ecran 4 - challenge - Boss de Session 4
- Question: "Resous ce signal complexe : 10 x (15 - 5) + 5"
- Reponse attendue: "105"
- Hint: "Parenthese, puis multiplication, puis addition."
- Feedback: "Precision chirurgicale !"

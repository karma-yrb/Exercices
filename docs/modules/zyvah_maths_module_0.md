## Mission 1 - Initiation : Regles de calcul

### Objectifs pedagogiques et competences
- Appliquer les regles de base avant de commencer les missions.

### Index technique (ordre des steps)
1. lesson - Regle 1 : Consigne jusqu'au bout
2. lesson - Regle 2 : Donnees claires
3. lesson - Regle 3 : Priorites
4. lesson - Regle 4 : Unites coherentes
5. lesson - Regle 5 : Test de logique

### Ecran 1 - lesson - Regle 1 : Consigne jusqu'au bout
- Texte: "Lis la consigne jusqu'au bout avant de calculer."

### Ecran 2 - lesson - Regle 2 : Donnees claires
- Texte: "Note les donnees (valeurs, unites, mots-cles) avant d'agir."

### Ecran 3 - lesson - Regle 3 : Priorites
- Texte: "Respecte l'ordre des priorites, parenthese d'abord."

### Ecran 4 - lesson - Regle 4 : Unites coherentes
- Texte: "Garde les unites coherentes avant de comparer."

### Ecran 5 - lesson - Regle 5 : Test de logique
- Texte: "Verifie si le resultat est logique dans la situation."

## Mission 1 - Le Souffle : La Virgule

### Objectifs pedagogiques et competences
- Utiliser la virgule pour separer des elements et des actions.
- Revoir le rythme d'une phrase par la ponctuation.

### Index technique (ordre des steps)
1. lesson - Briefing : Respiration Slam
2. interactive - Detection de Micro-coupure
3. write - Saisie Rythmique
4. challenge - Boss de Fin - Mission 1

### Ecran 1 - lesson - Briefing : Respiration Slam
- Texte: "Comme dans un texte de Gael Faye ou un morceau d'Oxmo Puccino, la virgule n'est pas qu'un signe technique, c'est ton souffle."
- Texte: "Elle sert a separer des elements dans une liste ou a marquer une courte pause sans couper le signal. C'est elle qui donne du groove a ta phrase."
- Exemple: "J'ai pris mon casque, mes gants et ma souris."
- Exemple: "Elle etait pale, elle marchait vite, elle semblait chercher quelqu'un."

### Ecran 2 - interactive - Detection de Micro-coupure
- Question: "Ou manque-t-il une virgule pour que la phrase soit fluide ?" (Le capitaine Lovyc analyse la map il deploie ses drones.)
- Options: "Apres 'Lovyc'" / "Apres 'map'" / "Apres 'deploie'"
- Reponse: "Apres 'map'"
- Feedback: "Correct. La pause se fait entre deux actions distinctes."

### Ecran 3 - write - Saisie Rythmique
- Question: "Reecris cette phrase en ajoutant les deux virgules necessaires :" (lag - sature - ping - le - montent - les - et - arrive - serveur - le)
- Requirements:
  - mode: "sentence"
  - keywords: ",", "serveur", "ping", "lag"
- Hint: "Il y a trois actions. Separe les deux premieres par des virgules."
- Feedback: "Bravo. Ton flow est desormais ininterrompu."

### Ecran 4 - challenge - Boss de Fin - Mission 1
- Question: "Redige une phrase listant 3 equipements tactiques (ex: radar, drone, radio) en utilisant correctement les virgules et le mot 'et' :"
- Requirements:
  - mode: "sentence"
  - keywords: ",", "et"
- Hint: "Objet 1, Objet 2 et Objet 3."
- Feedback: "Transmission recue. Ton inventaire est clair, Agent."

## Mission 2 - Dialogue & Incises

### Objectifs pedagogiques et competences
- Identifier et utiliser une incise.
- Construire une ligne de dialogue avec tiret cadratin.

### Index technique (ordre des steps)
1. lesson - Le Rapport de George Sand
2. interactive - Identification de Source
3. write - Codage de Dialogue
4. challenge - Boss de Fin - Mission 2

### Ecran 1 - lesson - Le Rapport de George Sand
- Texte: "Quand tu rapportes un dialogue, utilise le tiret cadratin (—) pour montrer qui parle. C'est la base de la precision litteraire."
- Texte: "L'incise (ex: dit l'agent, repond le coach) permet de savoir qui a transmis l'info sans alourdir le texte."
- Exemple: "— La zone est securisee, affirma Lovyc."

### Ecran 2 - interactive - Identification de Source
- Question: "Dans cette phrase, quelle partie est l'incise ?" (— Le patch arrive demain, murmura le developpeur.)
- Options: "— Le patch arrive demain" / "murmura le developpeur" / "le developpeur"
- Reponse: "murmura le developpeur"
- Feedback: "Exactement. L'incise donne le ton et l'auteur de la parole."

### Ecran 3 - write - Codage de Dialogue
- Question: "Complete ce dialogue avec une incise de ton choix (ex: s'exclama Zyvah, repondit le chef) :" (— Attention au sniper a gauche, _______.)
- Requirements:
  - mode: "sentence"
  - keywords: "Lovyc", "Zyvah", "chef", "agent", "dit", "disent", "repondit", "repondent", "s'exclama", "s'exclament", "hurla", "hurlent", "murmura", "murmurent"
- Hint: "Utilise un verbe de parole et un nom de personnage."
- Feedback: "Parfait. On sait maintenant d'ou vient l'alerte."

### Ecran 4 - challenge - Boss de Fin - Mission 2
- Question: "Ecris une courte ligne de dialogue commencant par un tiret cadratin (—) et finissant par une incise (ex: dit-il, repondit Lovyc) :"
- Requirements:
  - mode: "sentence"
  - keywords: "—"
- Hint: "N'oublie pas le tiret au debut et la virgule avant l'incise si necessaire."
- Feedback: "Dialogue valide. La communication entre agents est fluide."

## Mission 3 - L'Intensite (Points !?...)

### Objectifs pedagogiques et competences
- Choisir la ponctuation selon l'intention (ordre, question, suspens).
- Transformer une phrase en question.

### Index technique (ordre des steps)
1. lesson - Briefing : Signaux de Fin
2. interactive - Analyse de Signal
3. write - Nuance Tactique
4. challenge - Boss de Fin - Mission 3

### Ecran 1 - lesson - Briefing : Signaux de Fin
- Texte: "Le point final (.) est neutre. Mais en situation critique, tu as besoin de plus :"
- Liste: "! (Exclamation) : Pour l'urgence ou l'ordre." / "? (Interrogation) : Pour le doute ou le scan." / "... (Suspension) : Pour le suspens ou l'attente."
- Texte: "Attention a ne pas en abuser, sinon ton signal devient illisible."

### Ecran 2 - interactive - Analyse de Signal
- Question: "Quelle ponctuation choisirais-tu pour une alerte imminente ?"
- Options: "Le point simple (.)" / "Le point d'exclamation (!)" / "Les points de suspension (...)"
- Reponse: "Le point d'exclamation (!)"
- Feedback: "Correct. L'urgence demande une exclamation."

### Ecran 3 - write - Nuance Tactique
- Question: "Transforme cet ordre neutre en une question inquiète :" (Le drone est encore en vol.)
- Requirements:
  - mode: "sentence"
  - keywords: "?"
- Hint: "Change juste le point final par un point d'interrogation."
- Feedback: "Le doute est transmis. L'escouade va verifier."

### Ecran 4 - challenge - Boss de Fin - Mission 3
- Question: "Ecris une phrase d'action se terminant par des points de suspension (...) pour montrer que quelque chose arrive :"
- Requirements:
  - mode: "sentence"
  - keywords: "..."
- Hint: "Ex: Les ombres approchent doucement..."
- Feedback: "Suspens reussi. Tes troupes sont en alerte."

## Mission 4 - Precision Technique

### Objectifs pedagogiques et competences
- Utiliser deux-points pour introduire une liste.
- Inserer une precision avec parentheses.

### Index technique (ordre des steps)
1. lesson - Le Briefing Expert
2. interactive - Logistique de Phrase
3. write - Saisie de Precision
4. challenge - Boss de Fin - Mission 4

### Ecran 1 - lesson - Le Briefing Expert
- Texte: "Pour etre un maitre de la comm, utilise les outils des pros :"
- Liste: "Les deux-points (:) : Pour introduire une liste ou une explication." / "Les parentheses () : Pour ajouter une precision technique sans casser le rythme."
- Exemple: "Rapport : Lovyc a 3 items (rare, epique, legendaire)."

### Ecran 2 - interactive - Logistique de Phrase
- Question: "Que permettent d'introduire les deux-points (:) ?"
- Options: "Une pause pour respirer" / "Une liste ou une explication" / "Le nom de l'auteur"
- Reponse: "Une liste ou une explication"
- Feedback: "Exactement. C'est le signal de l'inventaire ou de la preuve."

### Ecran 3 - write - Saisie de Precision
- Question: "Ajoute une precision entre parentheses sur la map (ex: map de nuit) dans cette phrase :" (Nous arrivons sur la map.)
- Requirements:
  - mode: "sentence"
  - keywords: "(", ")"
- Hint: "Insere tes parentheses juste apres le mot 'map'."
- Feedback: "Donnee enregistree. L'escouade connait desormais les details."

### Ecran 4 - challenge - Boss de Fin - Mission 4
- Question: "Ecris une phrase utilisant les deux-points (:) pour lister deux choses necessaires a ta mission :"
- Requirements:
  - mode: "sentence"

- Feedback: "Logistique impeccable. Tu es pret pour le deploiement final."

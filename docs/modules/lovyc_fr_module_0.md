## Mission 1 - Initiation : Regles d'ecriture

### Objectifs pedagogiques et competences
- Appliquer les regles de base de l'ecrit avant de commencer le module.

### Index technique (ordre des steps)
1. lesson - Regle 1 : Majuscule
2. lesson - Regle 2 : Point final
3. lesson - Regle 3 : Une phrase = une idee
4. lesson - Regle 4 : Verbe obligatoire
5. lesson - Regle 5 : Pas de langage SMS

### Ecran 1 - lesson - Regle 1 : Majuscule
- Texte: "Commence chaque phrase par une majuscule."

### Ecran 2 - lesson - Regle 2 : Point final
- Texte: "Termine chaque phrase par un point."

### Ecran 3 - lesson - Regle 3 : Une phrase = une idee
- Texte: "Une phrase = une seule idee. Si tu as deux idees, fais deux phrases."

### Ecran 4 - lesson - Regle 4 : Verbe obligatoire
- Texte: "Chaque phrase doit contenir un verbe."

### Ecran 5 - lesson - Regle 5 : Pas de langage SMS
- Texte: "Pas de langage SMS dans les phrases."

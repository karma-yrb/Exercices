## Mission 1 - Initiation : Règles d'écriture

### Objectifs pédagogiques et compétences
- Appliquer les règles de base de l'écrit avant de commencer le module.

### Index technique (ordre des steps)
1. lesson - Règle 1 : Majuscule
2. lesson - Règle 2 : Point final
3. lesson - Règle 3 : Une phrase = une idée
4. lesson - Règle 4 : Verbe obligatoire
5. lesson - Règle 5 : Pas de langage SMS

### Écran 1 - lesson - Règle 1 : Majuscule
- Texte: "Commence chaque phrase par une majuscule."

### Écran 2 - lesson - Règle 2 : Point final
- Texte: "Termine chaque phrase par un point."

### Écran 3 - lesson - Règle 3 : Une phrase = une idée
- Texte: "Une phrase = une seule idée. Si tu as deux idées, fais deux phrases."

### Écran 4 - lesson - Règle 4 : Verbe obligatoire
- Texte: "Chaque phrase doit contenir un verbe."

### Écran 5 - lesson - Règle 5 : Pas de langage SMS
- Texte: "Pas de langage SMS dans les phrases."
